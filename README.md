# Onyxspire Management - Luxury Creator Management Website

A high-end, luxury website for Onyxspire Management, featuring elegant design, smooth animations, and comprehensive creator management services.

## Features

- **Modern Stack**: Built with Next.js 14, React, TypeScript, and Tailwind CSS
- **Luxury Design**: Premium color scheme with elegant typography and animations
- **Rotating Video Hero**: Dynamic video background with luxury lifestyle content
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: Framer Motion for elegant page transitions
- **Complete Pages**: Home, Services, Testimonials, Creators, FAQ, and Application

## Pages

1. **Home** - Hero section with rotating video montage, features, and stats
2. **Services** - Comprehensive breakdown of Management, Growth, and Planning services
3. **Testimonials** - Creator success stories with detailed case studies
4. **Creators** - Showcase of creator tiers and achievements
5. **FAQ** - Interactive accordion with frequently asked questions
6. **Application** - Comprehensive application form for new creators

## Color Scheme

- **Primary**: Black Espresso (#121212), Ash Graphite (#1E1F24)
- **Accent**: Bronze Gold (#B08D57), Soft Ivory (#F5EFE7)
- **Secondary Accent**: Merlot Wine (#5C1F2D)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Video Assets

The hero section uses rotating video backgrounds. The current video URLs in `components/VideoHero.tsx` are placeholders. Replace them with your actual luxury video assets:

- Models/fashion content
- Expensive luxury cars
- High-end laptops/technology
- Beach bungalows/resorts
- Private jets

Update the `videoSources` array in `components/VideoHero.tsx` with your video file paths or URLs.

## Build for Production

```bash
npm run build
npm start
```
