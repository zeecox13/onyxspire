# Brand Logos

Add your brand logo images here. Expected filenames:

- forbes.png
- oftv.png
- vogue.png
- vanity-fair.png
- gq.png
- elle.png
- harpers-bazaar.png
- ny-times.png
- wsj.png
- bloomberg.png

Logos should be white/transparent backgrounds and will be automatically inverted to white for display.

If logos are not available, the component will fall back to styled text.
