# Staying Motivated When Sales Slow Down

This folder contains images for the blog post "Staying Motivated When Sales Slow Down".

## Required Images

- `hero.jpg` or `hero.png` - Hero image used as the background in the blog post header

## Optional Images

Add any additional images you want to include in the blog post content here.

Images in this folder are referenced in the blog post content using paths like:
- `/blog/staying-motivated-when-sales-slow-down/hero.jpg`
- `/blog/staying-motivated-when-sales-slow-down/image-name.jpg`
