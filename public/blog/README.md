# Blog Post Images

This directory contains images for each blog post.

## Folder Structure

- `mental-health-adult-creator-industry/` - Images for "Mental Health in the Adult Creator Industry: Let's Talk About It"
- `what-creators-want-from-managers/` - Images for "We Asked 50 Creators What They Really Want From a Manager. Here's What They Said"
- `building-personal-brand/` - Images for "Building Your Personal Brand in the Creator Economy"

## Usage

Add images to the respective folder for each blog post. Images can be referenced in blog posts using paths like:
- `/blog/mental-health-adult-creator-industry/image-name.jpg`
- `/blog/what-creators-want-from-managers/image-name.jpg`
- `/blog/building-personal-brand/image-name.jpg`
